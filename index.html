Projects

Capstone

In my fourth and final year of my Bachelors, I led a team of four to create a thermal detection system for high body temperatures. This system used a thermal imaging camera and Artificial Intelligence to detect and caluclate a persons body temperature when entering the cameras view. We built a prototype using a Rasberry Pi and the FLIR Lepton thermal imaging camera. Using our python program, we incorporated UVC and the HOG (Histogram of Oriented Gradients) algorithm into our code which allowed for our program to detect objects when captured by the camera. Once a person was detected, it would use the radiometric values captured from the camera and export it to a NumPy Array to calculate the users overall temperature values. After the persons body temperature was calulated, if the temperature was above 37 degress celsius it would then screenshot that frame and be uploaded to our cloud hosted application. This project was thought of as a solution to help mitigate further transmission of COVID-19.
